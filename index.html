⚡ masukkan URL dan klik AMBIL KODE

✨ FUNGSI TOMBOL ✨

  • 🚀 AMBIL KODE - Mengambil kode HTML dari URL yang dimasukkan
  • ⏹️ BATAL - Membatalkan proses pengambilan kode dan menghapus tampilan kode
  • 🟢 SEMUA (Hijau) - Mode terpisah (tampil dalam 3 tab: HTML, CSS, JS)
  • 🔴 SEMUA (Merah) - Mode gabung (semua kode jadi satu dengan jarak antar bagian)
  • 📄 HTML - Menampilkan kode HTML saja
  • 🎨 CSS - Menampilkan kode CSS dari tag style
  • ⚡ JS - Menampilkan kode JavaScript dari tag script inline

Cara Kerja:
1. Masukkan URL di kolom "isi URL anda"
2. Klik AMBIL KODE untuk mengambil konten
3. Kode langsung muncul (tanpa efek mengetik)
4. Gunakan tombol mode untuk filter tampilan
5. Klik BATAL untuk menghapus kode dan berhenti
6. Klik SALIN KODE untuk copy ke clipboard
